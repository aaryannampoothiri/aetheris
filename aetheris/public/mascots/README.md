# Mascot Images

Please place your mascot PNG files in this directory with the following exact filenames:

- `optimus-prime.png` - Optimus Prime mascot
- `bumblebee.png` - Bumblebee mascot
- `wall-e.png` - Wall-E mascot
- `aetheris.png` - Aetheris mascot

## Recommended Image Specifications:
- Format: PNG with transparency
- Size: 256x256px to 512x512px (will be automatically resized to fit)
- Background: Transparent
- Orientation: Character facing forward or slightly to the side

The mascot will appear in the bottom-right corner of the screen with a subtle bounce animation.
